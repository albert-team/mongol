# Mongol

Minimalistic MongoDB helper for Node.js.

## FEATURES

- Auto-connect support
- Enhanced JSON Schema draft-4 support
- Database hook/trigger support
- Useful builtin hooks

## INSTALLATION

### Requirements

- Node.js >= 10
- MongoDB >= 3.6

### Instructions

With yarn

```bash
$ yarn add @albert-team/mongol
```

With npm

```bash
$ npm i @albert-team/mongol
```
